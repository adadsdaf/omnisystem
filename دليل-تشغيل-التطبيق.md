# دليل تشغيل نظام إتقان سوفت

## ▶️ الطريقة الأولى: تشغيل كموقع (متصفح)

### المتطلبات
- Node.js 18 أو أحدث → [nodejs.org](https://nodejs.org)
- pnpm → يُثبَّت تلقائياً بالأمر أدناه

### خطوات التشغيل

افتح **Command Prompt** أو **PowerShell** في مجلد المشروع، ثم اكتب:

```
npx pnpm install
```

ثم افتح نافذتين منفصلتين:

**النافذة الأولى — السيرفر:**
```
npx pnpm --filter @workspace/api-server run dev
```

**النافذة الثانية — الواجهة:**
```
npx pnpm --filter @workspace/pos-system run dev
```

افتح المتصفح على: **http://localhost:20639**

---

## 🖥️ الطريقة الثانية: تشغيل كتطبيق سطح مكتب (.exe)

### المتطلبات
- Node.js 18 أو أحدث
- اتصال بالإنترنت (للبناء الأول فقط)

### الخطوات

1. انقر مرتين على ملف **`BUILD-DESKTOP.bat`**
2. انتظر حتى ينتهي (5-10 دقائق في المرة الأولى)
3. ستجد ملف الإعداد في مجلد: `electron-app\release\`
4. انقر مرتين على **`OmniSystem Setup x.x.x.exe`** لتنصيب البرنامج
5. بعد التنصيب يظهر اختصار على سطح المكتب — افتحه مباشرة

## ❓ حل المشاكل الشائعة

**المشكلة:** `pnpm: command not found`
**الحل:** شغّل الأمر: `npm install -g pnpm`

---

**المشكلة:** الصفحة لا تفتح في المتصفح
**الحل:** تأكد أن كلا النافذتين (السيرفر والواجهة) تعملان بدون أخطاء

---

**المشكلة:** `Port already in use`
**الحل:** أعد تشغيل الجهاز أو أغلق البرنامج من Task Manager
